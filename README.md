# Event Calendar

A simple desktop calendar application built with Python and wxPython. Add and view events for any date.

## Features

- Interactive calendar widget
- Add events to specific dates
- View events for selected dates

## Requirements

- Python 3.7+
- wxPython

## Installation

1. Clone the repository:
 ```
   bash
   git clone https://github.com/saijashwanth2007-web/CS-jackfruit-problem.git
   cd CS-jackfruit-problem
   ```
